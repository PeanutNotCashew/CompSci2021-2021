What I Learned This Unit - Eleanor Fan

Unit Lessons

Lesson 1 This lesson was an introduction to HTML. We learned about the parts of the webpage, like the head and the body. We also learned about the header and paragraph tags.
Lesson 2 This lesson was mainly about formatting. We learned about in-line and block elements, as well as formatting tags like <em>. We also learned about list elements.
Lesson 3 In this lesson, we learned about how to insert weblinks and images into HTML sites. We also learned about attributes and comment tags.
Lesson 4 This lesson was mainly about text-based tags. We went more in-depth about headers and paragraph tags, and learned about the <pre> tag, which shows preformatted text. We also learned about Lorem Ipsum text.
Lesson 5 This lesson was about tables. We learned about table structure and the tags related to HTML tables.
Lesson 6 This lesson was about iframes. We learned how to embed external websites in HTML code.
Lesson 7 This lesson was about metadata. We learned and reviewed the types of tags that go in the head of a web page, like the <meta> tag.
Lesson 8 In this lesson, we learned how to put abnormal characters in the body of a website, like the < and > characters. We learned how to use unicode to do this.
Lesson 9 This lesson was about file paths. We learned how to add links to external HTML documents.
Lesson 10 This lesson was a re-hash of lesson 9.
Lesson 11 This lesson was about using enhanced text editors. We learned how to use Atom.
