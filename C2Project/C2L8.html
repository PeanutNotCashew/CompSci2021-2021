Unit C2 Lesson 8: Entities

Tags we learned this unit:

Tag Use
< a > Inserts a link
< body > Establishes the body of the webpage, where visible text is contained
< h1 > A tag for organizing text. Included in the category of heading tags, with < h2 >, < h3 >, etc.
< head > Estabishes the header of the webpage, where the metadata is contained
< img > Inserts a website
< img > Inserts an image
< li > Contains list data in < ol > or < ul >
< link > Contains metadata that links to other files, for example, CSS files
< meta > Contains metadata, including but not limited to character set and author name
< ol > Lists items in numerical order
< p > Placed in the body, it contains long blocks of text. Also known as the paragraph tag
< pre > Is similar to < p > tag in that it contains long blocks of text. Shows text as it is formatted in the html document
< script > Placed in the header, it allows for internal Javascript programming or links to external Javascript files
< style > Placed in the header, it allows for internal Cascading Style Sheets
< table > Creates a table for table data
< title > Contains metadata, specifically, the title of a document
< td > Contained in a table, it contains table data
< th > Contained in a table, it contains column headings
< tr > Contained in a table, it creates table rows
< ul > Lists items under bullet points
